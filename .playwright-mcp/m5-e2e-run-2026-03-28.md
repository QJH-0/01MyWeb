# M5 文件管理 E2E 执行摘要（2026-03-28）

## 环境

- `docker compose up -d mysql redis elasticsearch minio`（容器就绪）
- 后端：`mvn spring-boot:run`（`SPRING_PROFILES_ACTIVE=dev`，**8080**）；前端：`npm run dev`（**5173**）
- 账号：`myweb-admin` / `MyWebAdm1n2026`；`X-Admin-Token` 与 `frontend/.env` 中 `VITE_ADMIN_TOKEN`、后端 `APP_ADMIN_TOKEN` 一致
- 联调前置：**8080 须为当前分支构建的后端**。若旧 Java 进程长期占用 8080 且未含 M5，则 `/api/admin/files` 可能返回 `500 INTERNAL_ERROR`（本机已遇）；处理：结束占用 8080 的进程后重新 `spring-boot:run`。
