# 00D-M7 动效与体验优化：类设计蓝图（按需读取）

> 何时读取：仅当在 M7 阶段需要锁定前端模块职责边界时读取。

---

## 1. 实现方案（M7）

1. 页面切换动效标准化，禁止破坏可访问性。
2. 引入性能指标采集（LCP/INP/CLS）。
3. 统一前端反馈状态（toast/empty/error/loading）。

---

## 2. 前端模块职责（必须存在）

| 模块名 | 职责 | 类型 |
|--------|------|------|
| `usePageMotion()` | 页面切换动效 | Composable |
| `useUiFeedback()` | 统一反馈状态 | Composable |
| `reportWebVitals(metric)` | 性能指标上报 | Utility |
| `RouteTransition.vue` | 路由过渡组件 | Component |

说明：具体实现细节在 `07-M7` 阶段文档中定义。
