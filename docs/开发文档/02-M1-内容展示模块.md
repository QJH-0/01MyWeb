# M1 内容展示模块

## 阶段目标

完成网站基础信息展示页面，形成可访问、可浏览的门户骨架。

## 开发范围

- 页面：`/`、`/about`、`/experience`、`/contact`。
- 首页包含：个人简介、精选项目入口、最新博客入口、AI 入口占位。
- 联系页包含：静态联系方式 + 表单框架（先不接入完整发送链路）。

## 任务拆分

1. 搭建页面路由、导航与基础布局。
2. 构建静态数据模型（个人信息、经历、联系方式）。
3. 抽取公共组件（Section、Card、TimeLine、ContactForm）。
4. 实现响应式布局与基础样式变量。
5. 增加占位 API（后续可切换真实接口）。

## 验收清单

- 页面路由可达，导航高亮正确。
- 移动端与桌面端布局不破坏。
- 联系方式表单完成基础校验（必填、邮箱格式）。
- 首页入口可跳转至项目/博客/AI 功能位。

## 阶段完成定义（DoD）

- 完成 4 个页面联通与 UI 基线。
- 无阻断报错并通过基础回归（首页、导航、页内交互）。
- 通过后进入 M2。
