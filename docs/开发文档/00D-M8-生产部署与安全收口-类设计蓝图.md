## 2. 核心类职责（必须存在）

| 类名 | 职责 | 所属包 |
|------|------|--------|
| `SecurityConfig` | Spring Security 配置 | `security` |
| `JwtAuthenticationFilter` | JWT 解析与用户上下文注入 | `security` |
| `RateLimitFilter` | 接口限流 | `common` |
| `SecurityHeadersFilter` | 安全响应头 | `common` |
| `DeployVerificationService` | 部署后冒烟测试 | `service` |
